⚙️ 如何查詢處理器與記憶體
以下步驟適用於 Windows 10 / Windows 11，筆電與桌機皆適用。
🪟 方法一：透過「設定」查詢（最簡單）
1
點選左下角「開始」→「設定」（齒輪圖示）。
2
點選「系統資訊」或「關於」。
3
在「裝置規格」區塊中，即可找到以下資訊：
處理器（CPU） 例：Intel(R) Core(TM) i5-10400 → 填 i5-10400
已安裝 RAM 例：8.00 GB → 填 8GB
💻 方法二：透過「系統資訊」查詢
1
按下 Windows 鍵 + R，開啟「執行」視窗。
2
輸入 msinfo32，按 Enter。
3
在「系統資訊」視窗中，找到以下欄位：
處理器 例：Intel Core i5-10400 → 填 i5-10400
已安裝實體記憶體 例：8,192 MB → 填 8GB
⚠️ 填寫提醒：只需填寫核心型號即可（例如 i5-10400、i7-1165G7），不需要填完整名稱。若查無資訊請填「無」。
